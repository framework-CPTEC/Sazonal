CPTEC-SAZONAL Documentação
========================

É um pacote in Python para a distribuição de dados brutos de previsão sazonal do CPTEC/INPE  produzidos com o modelo BAM-1.2. Os arquivos de dados contêm a média do conjunto (de 15 membros) para todos os produtos e variáveis mostradas no site https://sazonal.cptec.inpe.br/.

Esses arquivos estão disponíveis como anomalias computadas referentes à climatologia 1981-2010.

.. note::

   support Python >= 3.10.
