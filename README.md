### CPTEC-SUBSAZONAL

É um pacote in Python para a distribuição de dados brutos de previsão subsazonal do CPTEC/INPE produzidos com o modelo BAM-1.2. Os arquivos de dados contêm a média do conjunto (de 11 membros) para todos os produtos e variáveis mostradas no site [https://subsazonal.cptec.inpe.br/](https://subsazonal.cptec.inpe.br/).

Esses arquivos estão disponíveis como anomalias computadas referentes à climatologia 1999-2018.

support Python >= 3.10.
